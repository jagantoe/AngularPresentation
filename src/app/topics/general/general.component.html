Angular releases a new version every 6 months.

Angular versions stay supported for 18 months after initial release. So 3 versions are always in support. The latest release will get new features and the other two will receive security and critical fixes. v9 - Ivy Compiler, v13 - IE11 dead, v14 - Standalone Components, v16 - Signals, v17 - New Control Flow, v19 - Hot module replacement, v20 - Zoneless + Stability.
